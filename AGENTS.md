# Project Instructions

## MT4 / MT5 Manager Safety Rule

- Strictly prohibit any operation in MT4 Manager or MT5 Manager other than data export.
- Allowed actions are limited to read-only inspection and exporting data required for analysis.
- Do not create, modify, delete, disable, enable, adjust, approve, reject, settle, transfer, close, open, or otherwise change any account, trade, order, group, symbol, configuration, permission, balance, credit, leverage, risk setting, or server-side state in MT4 Manager or MT5 Manager.
- If any non-export operation in MT4 Manager or MT5 Manager appears necessary, stop first and request explicit human review and approval from the user before taking action.
- Approval must be specific to the exact operation, target, and expected effect. General permission is not sufficient.
